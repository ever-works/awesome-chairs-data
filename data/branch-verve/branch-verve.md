# Branch Verve

**Category:** Chair Accessories  
**Brand:** Branch

## Overview
The Branch Verve is an ergonomic office chair designed for comfortable, all-day computer work. It offers several adjustment options aimed at supporting an upright, healthy posture.

## Features
- Ergonomic office chair for extended computer and desk work
- Designed to support an upright, straight-back posture
- Multiple adjustment options for personalized comfort and fit
- Provides cushioned, comfortable support for daily use

## Limitations / Considerations
- No seat tilt adjustment
- Armrests have relatively limited adjustability
- Seat is relatively narrow, which may not suit all body types
- Long-term use has revealed occasional cylinder (height mechanism) issues
