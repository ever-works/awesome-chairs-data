## ErgoIQ

![Humanscale Logo](https://www.humanscale.com/images/logo.png)

### Description
ErgoIQ is an ergonomic software tool developed by Humanscale designed to enhance user comfort and productivity by optimizing workstation setups. The software provides personalized recommendations tailored to individual needs and workspace configurations.

### Features
- **Personalized Recommendations**: Offers customized advice based on the user's specific ergonomic needs.
- **Workspace Configuration**: Adjusts recommendations according to the existing workspace setup.
