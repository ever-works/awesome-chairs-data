# BodyBilt Work From Home Solutions

Ergonomic work-from-home packages that combine seating, height-adjustable work surfaces, and monitor arms to create customizable home workstations for remote and hybrid employees.

Source: [BodyBilt – Work From Home Solutions](https://bodybilt.com/workspaces/work-from-home-solutions/)

---

## Features

### Ergonomic Workstation Package A
- Configure a complete ergonomic home workstation around the Sola LT chair.
- Component options:
  - **Sola LT ergonomic chair**
  - **Series 2 height-adjustable table**
  - **Keynote Core single or dual monitor arm**
- Purchase/configuration choices:
  - Chair only
  - Height-adjustable table only
  - Chair + height-adjustable table
  - Chair + height-adjustable table + single or dual monitor arm
- Packages are inclusive of freight costs.
- Can be shipped to either the employer’s workplace or directly to employees’ homes.

### Key Components

**Sola LT Chair**
- Ergonomic task chair used as the core seating option in Package A.
- Designed to provide a balance of comfort and value for extended computer use.

**Series 2 Height-Adjustable Table**
- Height-adjustable workstation table.
- Designed to encourage healthy repositioning during the workday.
- Supports positive movement for seated workers transitioning between postures.

**Keynote Core Monitor Arms**
- Flat-panel monitor arms for single or dual displays.
- Engineered to provide high flexibility for changing workstation setups.

**Maestro Sit-Stand Converter**
- Desktop solution that converts a fixed-height desk into a sit-stand workstation.
- Offers a range of sitting and standing heights.
- Designed to maintain aesthetics and usable workspace while adding adjustability.
